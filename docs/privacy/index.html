Privacy Policy

Last updated: January 31, 2026

Slack Mention Notifier is an open-source macOS application. Your privacy is important to us. This policy explains what data the app accesses and how it's handled.

What data does the app access?

The app connects to Slack's API to monitor messages in channels where the bot is a member. When a message mentioning you is detected, the app reads:

  • The message text
  • The sender's display name
  • The channel name
  • A permalink to the message

Where is data stored?

All data stays on your Mac. Specifically:

  • Slack tokens are stored in the macOS Keychain (encrypted by the operating system)
  • Reminders are created locally in Apple Reminders via EventKit
  • Configuration is stored in ~/.config/slack-mention-notifier/

The app does not send data to any third-party service, analytics platform, or external server. The only network communication is directly with Slack's API (slack.com).

No data collection

The app does not collect, transmit, or store any personal data beyond what is needed for its core functionality. There is no telemetry, no analytics, no tracking.

No server component

The app runs entirely on your Mac using Slack's Socket Mode (WebSocket). There is no server-side component. No data passes through any intermediary.

Open source

The entire source code is available at github.com/musketyr/slack-mention-notifier. You can audit exactly what the app does.
